## 🔄 HOW TO CHANGE AN IMAGE

### Step 1: Find New Image
Visit [unsplash.com](https://unsplash.com) or [pexels.com](https://pexels.com)

### Step 2: Get Direct URL
```
https://images.unsplash.com/photo-ID?w=WIDTH&h=HEIGHT&fit=crop&auto=format&q=80
```

### Step 3: Replace in HTML
Find the old image URL and replace with new URL

### Step 4: Update Alt Text
`alt="New description of image"`

### Example Replace
```html
<!-- OLD -->
<img src="https://images.unsplash.com/photo-1556909114..." alt="Old">

<!-- NEW -->
<img src="https://images.unsplash.com/photo-1234567890..." alt="New description" loading="lazy">
```

## 📐 IMAGE SIZING GUIDE

### **For Portfolio/Gallery**
```
Width: 1200px
Height: 1600px
Ratio: 4:5
Format: ?w=1200&h=1600&fit=crop&auto=format&q=80
```

### **For Service Cards**
```
Width: 600px
Height: 600px
Ratio: 1:1
Format: ?w=600&h=600&fit=crop&auto=format&q=80
```

### **For Team/Small**
```
Width: 300px
Height: 300px
Ratio: 1:1
Format: ?w=300&h=300&fit=crop&auto=format&q=80
Display: 120x120px in HTML
```

### Image Parameters
```
?w=1200          ← Set width
&h=1600          ← Set height
&fit=crop        ← Crop to ratio
&auto=format     ← Auto JPG/WebP
&q=80            ← Quality level
```

## 📞 TROUBLESHOOTING

### Image Not Loading?
1. Check URL is correct (copy from address bar)
2. Verify `w=` and `h=` parameters match
3. Clear browser cache (Ctrl+Shift+Delete)
4. Try in different browser

### Image Quality Poor?
1. Increase quality parameter: `&q=85` or `&q=90`
2. Ensure image resolution is 1200px+
3. Use original high-quality image

### Image Seems Cached?
1. Use Ctrl+Shift+R (hard refresh)
2. Clear browser cache entirely
3. Try incognito/private window
